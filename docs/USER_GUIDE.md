# Knowledge Base 使用指南

## 快速开始

### 1. 启动应用

```bash
# 启动 Docker 服务（PostgreSQL + Meilisearch）
pnpm docker:up

# 启动后端 API（终端1）
pnpm dev:api

# 启动前端 Web（终端2）
pnpm dev:web
```

### 2. 访问应用

| 页面 | URL | 说明 |
|------|-----|------|
| 首页 | http://localhost:3000 | 服务状态监控 |
| **文档库** | http://localhost:3000/documents | 文档管理主界面 |
| **编辑器** | http://localhost:3000/documents/new | 新建文档 |
| API 文档 | http://localhost:4000/api/docs | Swagger UI |
| 搜索引擎 | http://localhost:7700 | Meilisearch 控制台 |

> **重要**: 首页只显示服务状态，实际使用请访问 `/documents` 进入文档管理界面。

## 数学公式

### 行内公式

输入 `$E=mc^2$` 将渲染为行内公式。

### 块级公式

输入：
```
$$
\sum_{i=1}^n x_i = x_1 + x_2 + \cdots + x_n
$$
```

### 公式对话框

点击工具栏 ∑ 按钮打开公式插入对话框：

1. **选择模板**: 点击预设模板快速插入
2. **自定义输入**: 在输入框输入 LaTeX 语法
3. **实时预览**: 右侧实时显示渲染效果
4. **插入模式**: 选择行内或块级

### 常用 LaTeX 语法

| 语法 | 效果 |
|------|------|
| `\frac{a}{b}` | 分数 a/b |
| `x^{n}` | 上标 |
| `x_{n}` | 下标 |
| `\sqrt{x}` | 平方根 |
| `\sum_{i=1}^{n}` | 求和符号 |
| `\int_{a}^{b}` | 积分符号 |
| `\alpha, \beta, \gamma` | 希腊字母 |

## 表格编辑

### 创建表格

1. 点击工具栏表格按钮
2. 在弹出的网格中选择行列数
3. 点击确认插入

### 表格语法

```markdown
| 列1 | 列2 | 列3 |
| --- | --- | --- |
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |
```

### 表格对齐

| 语法 | 对齐方式 |
|------|----------|
| `:---` | 左对齐 |
| `:---:` | 居中 |
| `---:` | 右对齐 |

### WYSIWYG 编辑

表格在预览区可直接编辑：

- **编辑单元格**: 点击单元格直接输入
- **Tab 键**: 移动到下一个单元格
- **Shift+Tab**: 移动到上一个单元格
- **右键菜单**:
  - 在上方插入行
  - 在下方插入行
  - 在左侧插入列
  - 在右侧插入列
  - 删除行
  - 删除列
  - 设置对齐方式

## 图片管理

### 插入图片方式

1. **本地上传**: 点击图片按钮 → 本地上传 → 选择文件
2. **URL 链接**: 点击图片按钮 → URL 链接 → 输入图片地址
3. **拖拽上传**: 直接将图片拖入编辑器
4. **粘贴上传**: 复制图片后 Ctrl+V 粘贴
5. **图片库**: 从已上传的图片中选择（需先保存文档）

### 支持的图片格式

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)

### 图片大小限制

单张图片最大 **10MB**

### 图片存储位置

- 开发环境: `apps/api/uploads/images/`
- 访问路径: `/uploads/images/{filename}`

## 常见问题

### Q: 文档没有自动保存？

检查后端服务是否正常运行：
```bash
curl http://localhost:4000/api/v1/health
```

### Q: 图片上传失败？

1. 检查图片格式是否支持
2. 检查图片大小是否超过 10MB
3. 检查后端服务是否正常运行

### Q: 数学公式显示异常？

确保 LaTeX 语法正确，可使用公式对话框的实时预览功能验证。

### Q: 搜索找不到文档？

1. 确认 Meilisearch 服务正常运行
2. 等待索引更新（通常几秒内）
